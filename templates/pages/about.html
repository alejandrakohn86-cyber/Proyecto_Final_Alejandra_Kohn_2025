Sobre este proyecto

Este es mi proyecto final para el curso de Python en CoderHouse. Se trata de un sistema web desarrollado con Django que simula una plataforma de gestión empresarial con funcionalidades de blog.

Características del sistema

  • 📝 Sistema de gestión de clientes y productos
  • 📰 Blog con páginas y artículos
  • 👤 Sistema completo de autenticación (login, registro, perfiles)
  • 🖼️ Gestión de imágenes (avatares, fotos de páginas)
  • ✏️ Editor de texto enriquecido (CKEditor)
  • 🔒 Protección de rutas (solo usuarios logueados pueden editar/borrar)
  • 💬 Sistema de mensajería entre usuarios

Tecnologías utilizadas

  • Backend: Django 5.0, Python 3.x
  • Base de datos: SQLite3
  • Frontend: HTML5, CSS3, JavaScript
  • Librerías: django-ckeditor, Pillow
